# Ubuntu
```bash
sudo apt-get install libcsfml-dev libcsfml-doc 
```

In the root file run the command
```bash
gcc -o app *.c -lcsfml-graphics -lcsfml-system -lcsfml-window -lcsfml-audio -I./include
./app
```
